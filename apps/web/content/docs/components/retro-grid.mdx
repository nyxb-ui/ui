<TabsContent value="cli">

```bash
npx nyxbui@latest add retro-grid
```

Add the following animations to your tailwind.config.ts file:

```ts
// tailwind.config.ts
module.exports = {
  theme: {
    extend: {
      animation: {
        grid: "grid 15s linear infinite",
      },
      keyframes: {
        grid: {
          "0%": { transform: "translateY(-50%)" },
          "100%": { transform: "translateY(0)" },
        },
      },
    },
  },
};
```
